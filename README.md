# Paper Minecraft
HTML, CSS and Javascript 2D Minecraft game that can be downloaded or played in the browser. Not available for Android/IOS.

This is a 2D replica of the most popular sandbox game of all time: Minecraft. I have been one of Minecraft's biggest fans for the past few years and I thought it would be a fun idea to make a 2D version of it. I have tried to make it as much like Minecraft as possible.

I will regulary be posting updates here on this README.md file as I fix bugs and add new stuff. If you hve any complaints and suggestions please submit it in the Issues section.
